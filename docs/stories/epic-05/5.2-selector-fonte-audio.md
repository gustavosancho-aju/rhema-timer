# Story 5.2 — Selector de Fonte de Áudio na UI

**Epic:** EPIC-5 — Captura de Áudio via OBS
**Status:** Ready
**Prioridade:** Alta
**Estimativa:** 1h
**Depende de:** Story 5.1

## Contexto

Atualmente o app usa a fonte de áudio padrão do navegador. Precisamos que o operador escolha explicitamente qual dispositivo usar — microfone do notebook, CABLE Output (OBS), ou outro.

## User Story

**Como** operador de mídia
**Quero** selecionar na UI qual dispositivo de áudio usar antes de iniciar a gravação
**Para** alternar entre microfone e OBS sem precisar mexer nas configurações do sistema.

## Acceptance Criteria

- [ ] Antes de iniciar gravação, um dropdown mostra todos os dispositivos de entrada disponíveis
- [ ] Dropdown mostra nome do dispositivo (ex: "Microfone (Realtek)", "CABLE Output (VB-Audio)")
- [ ] Seleção do usuário é usada na captura (`SpeechRecognition` ou `getUserMedia`)
- [ ] Dispositivo selecionado é destacado visualmente quando ativo
- [ ] Se só 1 dispositivo disponível, dropdown é ocultado e usa ele direto
- [ ] Funciona mesmo depois do usuário conectar um dispositivo novo (listener de `devicechange`)

## Tarefas Técnicas

- [ ] Criar hook `useAudioDevices()` que retorna lista de `MediaDeviceInfo`
- [ ] Usar `navigator.mediaDevices.enumerateDevices()` filtrando por `kind === 'audioinput'`
- [ ] Requisitar permissão com `getUserMedia` antes (senão só retorna "Default")
- [ ] Adicionar listener `navigator.mediaDevices.addEventListener('devicechange', ...)`
- [ ] Criar componente `<AudioDeviceSelector>` em `src/components/audio-device-selector.tsx`
- [ ] Integrar no componente principal de gravação (`src/app/page.tsx`)
- [ ] Passar `deviceId` selecionado ao iniciar captura

## Referências técnicas

- **Web Speech API não aceita `deviceId` diretamente** — precisa rotear via `MediaStream` do `getUserMedia` e usar uma AudioContext intermediária, OU deixar o sistema escolher o default quando o usuário já tiver definido o CABLE Output como padrão do Windows.
- **Solução pragmática:** exibir o selector como informativo + orientar o usuário a definir o dispositivo como padrão no Windows em "Configurações de Som" (documentar em 5.1).
