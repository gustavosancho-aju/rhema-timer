# Story 7.3 — Timer Engine (Web Worker)

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Alta
**Depende de:** Story 7.1
**Estimativa:** 3 pontos

---

## Contexto

O coração do sistema. O timer deve continuar funcionando mesmo se a conexão WebSocket cair (requisito crítico do Stagetimer). Para isso, usamos um Web Worker isolado que mantém o loop de tick independente do thread principal. O servidor sincroniza `startedAt` + `duration`, e o cliente calcula o tempo restante localmente.

---

## Critérios de Aceitação

- [ ] Web Worker em `web/public/timer-worker.js` criado
- [ ] Worker recebe mensagem `{ type: 'start', startedAt, duration, timerType }` e começa a emitir ticks
- [ ] Worker emite `{ type: 'tick', elapsed, remaining, displayTime }` a cada 100ms
- [ ] Worker para ao receber `{ type: 'stop' }` ou `{ type: 'pause', pausedAt }`
- [ ] Worker retoma ao receber `{ type: 'resume', resumeAt }`
- [ ] Hook `useTimer(roomId)` em `web/src/hooks/timer/use-timer.ts`:
  - Conecta ao Web Worker
  - Retorna `{ displayTime, elapsed, remaining, status, isRunning }`
- [ ] Suporta os 3 modos:
  - `countdown`: conta de `duration` até 0
  - `countup`: conta de 0 em diante
  - `time_of_day`: exibe hora atual do dispositivo
- [ ] Ao reconectar WebSocket, sincroniza o estado atual do timer
- [ ] Store Zustand `useTimerStore` gerencia estado da sala ativa

---

## Protocolo WebSocket — Mensagens de Timer

```typescript
// Servidor → Cliente (broadcast para sala)
{ type: 'timer:start', timerId, startedAt, duration, timerType }
{ type: 'timer:pause', timerId, pausedAt, elapsed }
{ type: 'timer:resume', timerId, resumeAt }
{ type: 'timer:stop', timerId }
{ type: 'timer:sync', timerId, startedAt, elapsed, status }
{ type: 'room:state', timers: TimerState[], activeTimerId: string }
```

---

## Lógica do Cálculo de Tempo

```typescript
// countdown
remaining = duration - (Date.now() - startedAt) / 1000
displayTime = formatTime(Math.max(0, remaining))

// countup
elapsed = (Date.now() - startedAt) / 1000
displayTime = formatTime(elapsed)

// time_of_day
displayTime = new Date().toLocaleTimeString()
```
