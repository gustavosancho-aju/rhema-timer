# Story 7.7 — Interface Moderator

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Média
**Depende de:** Story 7.4
**Estimativa:** 2 pontos

---

## Contexto

O Moderator tem acesso apenas ao sistema de mensagens, sem controle de timers. É ideal para um integrante da equipe que só precisa enviar avisos ao apresentador sem ver ou interferir no controle de tempo.

---

## Critérios de Aceitação

- [ ] Rota `/timer/[roomId]/moderator` renderiza interface de mensagens
- [ ] Exibe o timer ativo no topo (somente leitura — tempo atual)
- [ ] Campo de texto para digitar mensagem
- [ ] Seleção de cor: Branco / Verde / Vermelho
- [ ] Botão Enviar → emite via WebSocket para todos os Viewers
- [ ] Histórico das últimas mensagens enviadas (sessão atual)
- [ ] Sem botões de play/pause/stop (somente visualização do timer)
- [ ] Indicador visual de quando mensagem foi recebida (confirmação)

## Protocolo WebSocket

```typescript
// Moderator → Servidor
{ type: 'message:send', roomId, text, color }

// Servidor → Todos (broadcast)
{ type: 'message:received', id, text, color, sentAt }
```
