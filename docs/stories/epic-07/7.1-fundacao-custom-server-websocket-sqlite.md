# Story 7.1 — Fundação: Custom Server + WebSocket + SQLite

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Crítica (bloqueante para todas as outras stories)

## Contexto

Antes de qualquer funcionalidade de timer, precisamos da infraestrutura base: um servidor Node.js customizado que substitui o servidor padrão do Next.js, adicionando suporte a WebSocket, e um banco SQLite com Drizzle ORM para persistência local.

## Schema do Banco (Drizzle/SQLite)

```typescript
// rooms
id: text (PK, nanoid)
name: text NOT NULL
created_at: integer (timestamp)
updated_at: integer (timestamp)

// timers
id: text (PK, nanoid)
room_id: text (FK → rooms.id)
title: text
presenter: text
type: text ('countdown' | 'countup' | 'time_of_day')
duration: integer (segundos)
scheduled_start: integer (timestamp, nullable)
color: text (label color, ex: 'red' | 'green' | 'blue' ...)
order: integer
status: text ('idle' | 'running' | 'paused' | 'finished')
started_at: integer (timestamp, nullable)
created_at: integer (timestamp)
updated_at: integer (timestamp)

// messages
id: text (PK, nanoid)
room_id: text (FK → rooms.id)
text: text NOT NULL
color: text ('white' | 'green' | 'red')
created_at: integer (timestamp)

// connections (log de dispositivos)
id: text (PK, nanoid)
room_id: text (FK → rooms.id)
type: text ('controller' | 'viewer' | 'operator' | 'moderator' | 'agenda')
label: text
connected_at: integer (timestamp)
```

---

## Protocolo WebSocket (mensagens JSON)

```typescript
// Cliente → Servidor
{ type: 'join', roomId: string, clientType: string }
{ type: 'ping' }

// Servidor → Cliente
{ type: 'pong' }
{ type: 'joined', roomId: string }
{ type: 'error', message: string }
```

## Notas de Implementação

- O `server.ts` deve criar um `http.Server` wrapping o Next.js handler, e anexar o `WebSocket.Server` nesse mesmo servidor HTTP.
- Usar `ts-node` para rodar o custom server em desenvolvimento.
- O banco deve ser inicializado automaticamente na primeira execução (migrations auto-apply em dev).
- WebSocket path: `ws://localhost:3000/ws/timer`
