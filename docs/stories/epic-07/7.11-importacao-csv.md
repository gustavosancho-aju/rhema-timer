# Story 7.11 — Importação de Agenda via CSV

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Baixa
**Depende de:** Story 7.2
**Estimativa:** 2 pontos

---

## Contexto

Permite criar múltiplos timers de uma vez via arquivo CSV (Excel/Google Sheets), replicando a funcionalidade do Stagetimer. Ideal para eventos com muitos itens na agenda.

---

## Critérios de Aceitação

- [ ] Botão "Importar CSV" na página de gerenciamento de salas ou no Controller
- [ ] Modal com instruções do formato esperado + link para template
- [ ] Upload do arquivo CSV via input file
- [ ] Parser valida colunas obrigatórias: `title`, `duration`
- [ ] Colunas opcionais: `presenter`, `type`, `color`, `notes`
- [ ] Preview dos timers que serão criados antes de confirmar
- [ ] Ao confirmar, cria todos os timers no banco preservando a ordem
- [ ] Tratamento de erros: linhas inválidas mostradas com motivo
- [ ] Template CSV disponível para download em `/public/timer-template.csv`

---

## Formato CSV esperado

```csv
title,presenter,duration,type,color
Abertura,Rev. Carlos,600,countdown,green
Louvor,,1200,countdown,blue
Palavra,Pastor João,2400,countdown,red
Ofertas,,300,countdown,yellow
Avisos,,600,countdown,white
```

- `duration`: em segundos (600 = 10 minutos)
- `type`: `countdown` | `countup` | `time_of_day` (default: `countdown`)
- `color`: `red` | `green` | `blue` | `yellow` | `white` | `purple` | etc.
