# Story 7.9 — Comportamentos Avançados de Timer

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Média
**Depende de:** Stories 7.3, 7.4, 7.5
**Estimativa:** 4 pontos

---

## Contexto

Além do play/pause/stop básico, o Stagetimer oferece comportamentos automáticos que tornam o sistema profissional: auto-avanço entre timers, início agendado, cores de alerta progressivas (wrap-up), alertas sonoros em marcos de tempo, e ajuste fino de tempo durante a apresentação.

---

## Critérios de Aceitação

### Auto-Advance
- [ ] Cada timer tem configuração `autoAdvance: boolean`
- [ ] Quando timer chega a 00:00, inicia automaticamente o próximo
- [ ] No Controller, toggle "Auto-Advance" visível por timer
- [ ] Broadcast `timer:start` do próximo timer é emitido pelo servidor

### Wrap-up Colors (Cores de Alerta)
- [ ] Cada timer tem `wrapupAt: number[]` (ex: `[300, 60, 0]` = 5min, 1min, fim)
- [ ] Viewer muda cor de fundo/barra progressivamente:
  - Verde (normal)
  - Amarelo quando atinge primeiro `wrapupAt`
  - Vermelho quando atinge segundo `wrapupAt`
  - Piscando quando tempo = 0 (opcional)
- [ ] Configurável no formulário de edição do timer

### Wrap-up Chimes (Alertas Sonoros)
- [ ] Arquivos de áudio curtos (beep) incluídos em `/public/sounds/`
- [ ] Chime toca no Viewer/Operator quando tempo atinge um `wrapupAt`
- [ ] Configurável: on/off por timer
- [ ] Usa Web Audio API (sem elemento `<audio>` bloqueado por autoplay)

### Time Warp (Ajuste de Ritmo)
- [ ] No Controller, botões `+1min` / `-1min` ajustam duração em tempo real
- [ ] Atualização propagada via WebSocket para todos os clientes
- [ ] Também permite `+10s` / `-10s` (Nudge)

### Scheduled Start (Início Agendado)
- [ ] Timer pode ter `scheduledStart: timestamp`
- [ ] Servidor inicia automaticamente quando horário chegou
- [ ] No Controller, campo "Iniciar às HH:MM" no formulário do timer
- [ ] Agenda exibe horário previsto de cada item

---

## Protocolo WebSocket — Novos Eventos

```typescript
// Servidor → Clientes
{ type: 'timer:wrapup', timerId, level: 0|1|2 }
{ type: 'timer:adjusted', timerId, newDuration }
{ type: 'timer:chime', timerId }
```
