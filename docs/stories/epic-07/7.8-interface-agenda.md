# Story 7.8 — Interface Agenda

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Média
**Depende de:** Story 7.2
**Estimativa:** 2 pontos

---

## Contexto

A Agenda exibe o cronograma completo do evento em formato de lista, mostrando todos os timers/itens com seus títulos, apresentadores e durações. É uma interface somente leitura, útil para compartilhar com a equipe ou exibir nos bastidores.

---

## Critérios de Aceitação

- [ ] Rota `/timer/[roomId]/agenda` renderiza cronograma completo
- [ ] Lista todos os timers da sala em ordem
- [ ] Cada item exibe: título, apresentador (se houver), duração formatada
- [ ] Item ativo (timer rodando) destacado visualmente
- [ ] Itens finalizados marcados com ✓
- [ ] Tempo total do evento calculado e exibido no footer
- [ ] Atualiza em tempo real quando o Controller avança para o próximo timer
- [ ] Sem controles (somente leitura)
- [ ] Responsivo (funciona bem em celular para equipe consultar)
- [ ] Se sala não existe, exibe mensagem de erro amigável
