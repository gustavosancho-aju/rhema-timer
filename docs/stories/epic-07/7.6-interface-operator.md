# Story 7.6 — Interface Operator

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Média
**Depende de:** Story 7.4
**Estimativa:** 2 pontos

---

## Contexto

O Operator é uma versão simplificada do Controller, otimizada para tablet. Exibe apenas os controles essenciais (play/pause/stop) sem os formulários de edição, sendo ideal para um assistente de palco operar durante o evento.

---

## Critérios de Aceitação

- [ ] Rota `/timer/[roomId]/operator` renderiza interface simplificada
- [ ] Layout otimizado para toque (botões grandes, espaçamento generoso)
- [ ] Exibe o timer ativo em destaque (tempo + título + apresentador)
- [ ] Controles visíveis: ▶ Play / ⏸ Pause / ⏹ Stop / ↺ Reset
- [ ] Lista de todos os timers da sala abaixo, com botão para tornar ativo
- [ ] Campo simples de mensagem (enviar ao Viewer)
- [ ] Sem formulários de edição (somente leitura da estrutura)
- [ ] Sincronizado em tempo real via WebSocket
- [ ] Funciona bem em tablet (iPad, Android 10")
