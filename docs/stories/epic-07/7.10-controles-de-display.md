# Story 7.10 — Controles de Display (Blackout, Flash, Focus Mode)

**Epic:** 07 — Timer de Apresentação
**Status:** Ready for Review
**Prioridade:** Média
**Depende de:** Stories 7.4, 7.5
**Estimativa:** 2 pontos

---

## Contexto

O Controller precisa de controles rápidos para manipular o que o apresentador vê no Viewer sem alterar o timer em si: apagar a tela completamente (Blackout), piscar para chamar atenção (Flash), ou esconder o timer e mostrar só a mensagem (Focus Mode).

---

## Critérios de Aceitação

### Blackout
- [ ] Botão "Blackout" no Controller (toggle on/off)
- [ ] Quando ativo: Viewer fica completamente preto
- [ ] Timer continua rodando internamente (não é pausado)
- [ ] Ícone/botão muda para indicar estado ativo
- [ ] Broadcast: `{ type: 'display:blackout', active: boolean }`

### Flash
- [ ] Botão "Flash" no Controller (momentâneo, não toggle)
- [ ] Viewer pisca a tela por ~500ms (branco ou colorido)
- [ ] Útil para chamar atenção do apresentador
- [ ] Broadcast: `{ type: 'display:flash' }`

### Focus Mode
- [ ] Botão "Focus" no Controller (toggle on/off)
- [ ] Quando ativo: Viewer esconde o timer e exibe apenas a mensagem atual em destaque
- [ ] Se não há mensagem, exibe tela vazia (quase blackout)
- [ ] Broadcast: `{ type: 'display:focus', active: boolean }`

### No Controller
- [ ] Três botões agrupados na área inferior do painel esquerdo
- [ ] Estado persistido no Zustand store
- [ ] Estado sincronizado ao entrar na sala (via `room:state`)

## Protocolo WebSocket — Novos Eventos
